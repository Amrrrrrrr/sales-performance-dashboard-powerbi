## Overview

Interactive sales analytics dashboard built using Power BI to analyze orders, revenue, tax, and product performance across territories and categories.

## Tools Used

* Power BI
* Power Query (ETL)
* DAX
* Data Modeling (Star Schema)
* Figma (UI Design)

## Data Model

Star schema consisting of:

* Fact Table: Orders
* Dimension Tables: Product, Territory, Status, Shipping

## Key KPIs

* Total Orders
* Total Revenue
* Total Tax
* Total Freight
* Order Quantity
* Customer Growth

## Insights

* Bikes category generates the highest revenue
* Canada and Northwest are top performing territories
* Order peak observed in March and October
